README.txt for use with PA1.c -- Przemek Gardias

To compile the program, enter the following command: gcc –Wall –o PA1 PA1.c

The output will be a file called PA1 in the directory in which PA1.c is in.

To run the program, enter the following command: ./PA1

The program will run and show the following output:
MONTHLY CALENDAR
Please enter year for this calendar:-

Type in any positive integer for the year, hit enter, and the provided year's calendar will be printed.
